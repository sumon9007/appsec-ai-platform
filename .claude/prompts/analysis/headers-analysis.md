# Headers Analysis

## Objective

Evaluate HTTP security headers for the target website.

## Inputs

Target URL  
Response headers  

## Instructions

Review the following headers:

Strict-Transport-Security  
Content-Security-Policy  
X-Frame-Options  
X-Content-Type-Options  
Referrer-Policy  
Permissions-Policy  

Determine:

- missing headers
- misconfigurations
- potential risks

## Output

Produce structured findings including:

- header issue
- risk explanation
- severity
- remediation

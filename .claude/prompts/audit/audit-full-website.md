# Full Website Security Audit

## Objective

Perform a structured security audit of the target website.

## Instructions

1. Read the context files.
2. Apply the relevant rules.
3. Use the following skills:

headers-tls-audit  
auth-access-audit  
rbac-audit  
input-validation-audit  
security-misconfig-audit  
logging-monitoring-audit  

4. Review evidence sources.

5. Record findings.

## Output

Produce:

- findings summary
- identified vulnerabilities
- review gaps
- prioritized next actions
